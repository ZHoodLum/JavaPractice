# JavaPractice

JDK JRE 区别
--
* java垃圾回收机制:
> 1、java垃圾回收  是自动释放该内存快

> 2、垃圾回收机制作用于堆内存当中

> 3、没有引用指向的对象 ，会被标记为垃圾

* 两种提示垃圾回收机制：

> 1）System.gc();  提醒系统进行垃圾回收（至于效率、怎末回收，无权干涉）

> 2）.Finalize() ：当某个对象被系统收集为无用信息的时候,finalize()将被自动调用,但是jvm不保证finalize()一定被调用,也就是说,finalize()的调用是不确定的,这也就是为什么sun不提倡使用finalize()的原因
